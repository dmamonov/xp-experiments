REST API Playground
===================
Examples of using the XP Framework's `webservices.rest` API.

Demo
----
To try out the [GitHub API](http://developer.github.com/) to retrieve issues
for a given user's project, execute:

```sh
$ xpcli cmd.github.IssuesOf user repo [-v]
```
